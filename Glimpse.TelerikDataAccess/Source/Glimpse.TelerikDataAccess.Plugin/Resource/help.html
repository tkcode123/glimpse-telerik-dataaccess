Glimpse DataAccess - Plugin module for Telerik DataAccess

The Glimpse DataAccess module provides improved insights into some usage metrics of Telerik DataAccess.
It is meant as a replacement for Glimpse.ADO for applications using Telerik DataAccess and provides some added metrics like fetch time (the time the ADO reader is active) and second level cache operations.

Usage

The following screen shots show the important metrics and measurements.



Profiled data is displayed in the timeline together with the connection identifier.



With the additional Download button you persist the trace data into a local file on the server (into the servers working directory). This can be useful for later offline analyzation or comparison.
